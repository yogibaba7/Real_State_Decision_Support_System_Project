# 🏠 Real Estate Decision Support System

🔗 **Live App:**
https://realstatedecisionsupportsystemproject.streamlit.app/

---

## 🚀 Overview

This project is a **Real Estate Decision Support System** built using **Machine Learning + Streamlit**.
It helps users analyze properties, understand price drivers, and make better buying decisions.

Instead of just showing data, the app provides **insights, comparisons, and intelligent recommendations**.

---

## ✨ Key Features

### 📊 1. Analytics Dashboard

* Zone-wise analysis
* Colony-wise insights
* Price vs Area trends
* Bedroom & furnishing distribution
* Amenities analysis

---

### 🧠 2. Insight Module

* Feature-wise impact on price
* Example:

  * Increasing area → price increase
  * More bedrooms → price change
* Designed like an **interactive decision engine**

---

### 🏠 3. Property Recommender

* Suggests similar properties based on:

  * Price similarity
  * Area similarity
  * Amenities similarity

---

### 💰 4. Price Prediction

* ML model predicts property price
* Based on:

  * Area
  * Bedrooms
  * Location
  * Amenities

---

### 📍 5. Zone Intelligence

* Manual + data-driven zoning
* Helps users understand:

  * Premium areas
  * Budget-friendly zones

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **ML Libraries:** scikit-learn, statsmodels
* **Visualization:** Plotly, Matplotlib, Seaborn
* **Data Processing:** Pandas, NumPy
* **Model Handling:** Joblib

## ⚙️ How to Run Locally

```bash
# clone repo
git clone <your-repo-link>

# go to folder
cd project-folder

# create venv
python -m venv venv

# activate
venv\Scripts\activate

# install dependencies
pip install -r requirements.txt

# run app
streamlit run app.py
```
